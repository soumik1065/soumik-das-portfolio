Università Ca' Foscari Venezia

Department: Dipartimento di Scienze Ambientali, Informatica e Statistica

Duration:01/10/2025 - 17/12/2025

  • Project Entitled: "A Drone Swarm Protocol for GPS Spoofing Detection"
  • Project Description: This project aims to developed a lightweight and fully distributed GPS spoofing detection protocol for UAV swarms operating in FANETs. The system combines physical-layer verification using RSSI-based trilateration and AoA-based triangulation with Practical Byzantine Fault Tolerance (PBFT) consensus to collaboratively identify spoofed drones. The approach eliminates dependence on a central controller or machine learning models, reduces false positives, and is suitable for real-time deployment on resource-constrained UAV platforms. Validated through multi-UAV simulations with dynamic mobility.
  • Supervisor: Prof. Agostino Cortesi, Professor, DAIS, Università Ca Foscari Venezia

Research Intern

Jadavpur University

Location: Jadavpur University Salt Lake Campus, Kolkata – 700106, West Bengal

Department: School of Mobile Computing and Communication

Duration:31/07/2023 - 31/08/2025

  • Project Entitled: "Path planning for drones flying beyond visual line of sight (BVLoS)"
  • Project Description: This project aims to improve the safety and efficiency of UAV operations conducted beyond the operator’s direct visibility, which are vital for applications such as infrastructure inspection, agriculture, disaster management, and delivery services. The work focuses on developing a risk-based path planning framework that generates grid-based risk maps by considering factors like no-fly zones, population density, sheltering conditions, obstacle heights, and probabilities of crash, impact, and fatality. Implemented through simulations in urban, sub-urban, and rural regions using a C program, the study revealed that the sheltering factor is the most critical parameter, with higher shelter values (e.g., buildings) reducing risks, while open areas exhibit the highest risks. The results show that effective BVLoS operations require not only robust navigation technologies but also comprehensive risk evaluation strategies. This project provides a foundation for advancing safer BVLoS drone missions, with future work focusing on dynamic risk assessment under real-time conditions.
  • Supervisor: Prof. Punyasha Chatterjee, Professor and Director, SMCC, Jadavpur University
